About arena-club

Arena Club is a place for the Soul fighter event launched by Riot games. When you have joined this club, you will be able to interact with other people that enjoy the Arena game mode that riotgames have created. We are currently working on a Discord server that will create a space for us players to enjoy this game mode together and hopfully find some new friends and premades!
